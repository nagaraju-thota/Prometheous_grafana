# Prometheus & Grafana Zero to Hero

A hands-on 7-day learning series covering monitoring concepts, Prometheus, Grafana, Kubernetes monitoring, dashboards, alerting, and a production-ready monitoring stack.

## Learning Plan

- Day 1 - Monitoring Fundamentals & Prometheus Basics
- Day 2 - Install Prometheus & Grafana on Kubernetes
- Day 3 - Kubernetes Cluster Monitoring
- Day 4 - Application Monitoring
- Day 5 - Grafana Dashboards
- Day 6 - Alerting with Alertmanager
- Day 7 - Real-World Monitoring Stack

Each day includes:
- 📘 Theory (`notes.md`)
- 🛠 Hands-on (`hands-on.md`)
- 💻 Commands (`commands.md`)
- 🏗 Architecture (`architecture.md`)
- 📸 Screenshots (`images/`)
