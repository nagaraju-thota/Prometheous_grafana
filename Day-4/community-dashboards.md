# Community Dashboards

Grafana provides ready-made dashboards that can be imported using dashboard IDs.

## Popular Dashboards

| Dashboard | ID |
|-----------|----|
| Node Exporter Full | 1860 |
| Kubernetes Cluster Monitoring | 315 |
| Kubernetes Views Global | 15757 |

## Import Steps

1. Open **Dashboards**.
2. Click **Import**.
3. Enter the dashboard ID.
4. Select the Prometheus data source.
5. Click **Import**.

Using community dashboards saves time and provides production-ready monitoring dashboards.
