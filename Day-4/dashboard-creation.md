# Creating a Grafana Dashboard

## Steps

1. Open Grafana.
2. Add Prometheus as the data source.
3. Create a new dashboard.
4. Add visualization panels.
5. Use PromQL queries to display metrics.
6. Save the dashboard.

## Dashboard Panels

- CPU Usage
- Memory Usage
- Running Pods
- Running Deployments
- Network Usage

This dashboard provides a quick overview of the Kubernetes cluster's health and resource usage.
