# Day 1 - Monitoring Fundamentals & Prometheus Basics

## Goal

Understand the fundamentals of monitoring and learn the basics of Prometheus.

---

## Topics Covered

- What is Monitoring?
- Monitoring vs Logging vs Tracing
- Metrics
- Time Series Database (TSDB)
- What is Prometheus?
- Prometheus Architecture
- Prometheus Components
- Prometheus Server
- Exporters
- Alertmanager
- Pushgateway
- Pull vs Push Model
- Metric Types
  - Counter
  - Gauge
  - Histogram
  - Summary

---

## Hands-on

- Pull Prometheus Docker image
- Run Prometheus locally
- Access Prometheus UI
- Explore built-in metrics
- Execute basic PromQL queries

---

## Outcome

By the end of this day you will understand the core concepts behind monitoring and Prometheus and be able to run Prometheus locally.
